# 图片资源

请在此目录放置以下图标文件（建议尺寸 81x81 px）：

- `home.png` - 首页图标
- `home-active.png` - 首页选中图标
- `book.png` - 图书图标
- `book-active.png` - 图书选中图标
- `user.png` - 用户图标
- `user-active.png` - 用户选中图标
- `default-cover.png` - 默认图书封面（建议 200x260 px）
- `default-avatar.png` - 默认头像（建议 96x96 px）

可以从 https://www.iconfont.cn/ 下载适合的图标。
